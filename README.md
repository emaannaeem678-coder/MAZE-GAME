AI Maze Solver Game
Description

AI Maze Solver is a Java Swing based desktop application that allows users to play and solve mazes using different game modes.

The project includes:

User Login & Registration
Welcome Screen
Level Selection
Manual Maze Solving
AI Maze Solver using BFS Algorithm
AI vs User Competition Mode
Multiple Maze Levels
Interactive GUI using Java Swing

The project demonstrates:

Graph Traversal Algorithms
Breadth First Search (BFS)
Java Swing GUI
Event Handling
Object Oriented Programming
Features
User Authentication
Login System
Registration System
Username & Password Validation
Levels
Level 1 → 4x4 Maze
Level 2 → 8x8 Maze
Level 3 → 12x12 Maze
Game Modes
1. Manual Mode

User solves maze manually using keyboard arrow keys.

2. AI Solver Mode

AI automatically solves maze using BFS algorithm.

3. AI vs User Mode
User solves maze manually
AI finds shortest path
Winner is decided based on number of steps

java AIMazeSolver
